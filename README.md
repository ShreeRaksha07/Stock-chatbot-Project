# 📊 Stock Market AI Chatbot

## 🚀 Project Overview
This is a beginner-friendly AI chatbot built using Python that helps users understand the stock market. It provides explanations, stock data, charts, and latest market insights.

---

## 🛠️ Technologies Used
- Python
- Streamlit (for UI)
- Ollama (Local LLM)
- Phi-3 / TinyLlama model
- yfinance (for stock data)

---

## ✨ Features
- 💬 Chat-based interface
- 📈 Stock price (AAPL, TSLA, etc.)
- 🧠 AI explanations for beginners
- 📰 Basic stock news insights
- 📊 Market trend explanation (Green/Red)

---

## ▶️ How to Run

1. Open terminal
2. Navigate to project folder:
    cd "Stock chatbot"
3. Run:
    streamlit run app.py
